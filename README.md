lua-tuntap
==========

Access TUN/TAP interfaces from lua.
Primarly developed for FreeBSD.
Experimental linux support.

Installation
============

Latest Git Revision
-------------------

With LuaRocks 2.1.2:

	$ luarocks install https://raw.githubusercontent.com/Vger/lua-tuntap/master/rockspecs/lua-tuntap-scm-0.rockspec
